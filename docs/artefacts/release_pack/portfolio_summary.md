# Portfolio Summary

The Graph-Based AML Case Triage and Risk Scoring System is a local-first financial crime analytics project that turns reference transactional data into alerts, graph features, risk scores, cases, validation artefacts, governance records, and reviewer-ready dashboard views.

## Portfolio Value

The project demonstrates end-to-end AML analytics engineering, model risk documentation, auditability, governance inventory, and security control packaging for `portfolio_release_v1`.
